# Titanic Data Cleaning Project

## Objective
Clean the Titanic dataset using Python and Pandas.

## Steps Performed
- Checked duplicate rows
- Identified categorical and numerical columns
- Checked unique values
- Calculated missing value percentages
- Removed unnecessary columns
- Filled missing values
- Detected outliers using Box Plot
- Removed outliers using Mean ± 2*STD
- Prepared dataset for Machine Learning

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib

## Dataset
Titanic Dataset
